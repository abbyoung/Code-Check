About

Why Code Check?

“The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect.”

—Tim Berners-Lee, W3C Director and inventor of the World Wide Web


The web was born accessible, and it's our job to keep it that way. Code Check was created to make web accessibility easier to achieve. With an initial focus on screen readers, Code Check utilizes Section 508 standards, screen reader habits, and current web standards to optimize html for screen reader users.

What's a screen reader?

A screen reader is software that interprets a computer screen's output and reads it to users. By writing semantic html, keeping current web standards in mind, and understanding how this technology parses our pages, developers can improve web accessibility for screen reader users and beyond.
